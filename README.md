# io-timestamp-peek

Requirements:
  NodeJs (0.10.*)

## Initialize
1. `git clone git@github.com:jpero09/io-timestamp-peek.git && cd io-timestamp-peek`
2. `npm install`

## To build dev assets
`grunt dev`

## To watch assets for changes and rebuild
`grunt watcher`

## To build prod assets
`grunt`

## To run node app
`node ./src/server/app.js`

## To view app
[http://localhost:3000](http://localhost:3000)
